# WolfISMS

Ein Informationssicherheitsmanagementsystem (ISMS) als Docker-Container.

## Installation

```bash
git clone <repo-url>
cd WolfISMS
cp .envdefault .env
```

`.env` bearbeiten und Werte anpassen.

Falls benötigt, Zertifikate nach `certs/` kopieren.

```bash
docker compose up -d
```

## Voraussetzungen
